# DictoGo

[English](README.md)

DictoGo 是一款面向英语学习者的听力与沉浸式学习 App，帮助你把真实内容变成每天可执行的英语练习。

它不是让你只背孤立单词，而是通过完整句子、真实语境、反复听读、词汇复习和句子输出练习，把“看得懂、听得懂”逐步转化成“说得出、用得上”。

## DictoGo 能帮你做什么

- 在真实英语内容中练听力、阅读和表达
- 用句子级翻译理解完整句子的结构和语境
- 从你真正读过、听过的内容里积累词汇
- 用生词本和单词训练复习收藏或导入的词汇
- 把值得反复练的句子加入句库，用来听、跟读和输出
- 用 Auto Echo 逐句跟读，训练口语节奏和表达反应
- 通过 AI 对话练习，把被动理解转成主动输出
- 记录每日学习进度，让坚持变得可见

## 核心功能

### 句子级翻译

理解完整句子，而不是只查单个单词。适合那些“单词大概认识，但整句话还是不顺”的学习场景。

### Auto Echo 跟读

按句子重复播放和跟读，把听力输入变成口语练习。

### 生词本与背单词

从真实内容里收藏或导入词汇，整理到生词本，再通过单词训练复习。词汇不是孤立死背，而是和你读过、听过的内容绑定。

### 句库

把文章里的好句子加入句库，也可以手动添加长文本并自动拆句。句库适合反复听、跟读、复盘表达，把“见过的句子”变成“能用的表达”。

### AI 对话练习

通过 AI 对话，把理解过的表达真正用出来。

### 沉浸式听读

从更接近真实使用场景的内容中学习，而不是只靠割裂的教材例句。

## 适合谁

DictoGo 适合这些英语学习者：

- 能看懂一点、听懂一点，但开口容易卡住
- 想通过真实语境学习英语
- 需要整句解释，而不是只查词
- 想把词汇复习和真实内容绑定起来
- 想把好句子收集起来，反复听、跟读和输出
- 不想死背长词表，更希望稳定每天练
- 想找到从输入到输出的清晰路径
