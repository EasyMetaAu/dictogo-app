# DictoGo

[English](README.md)

DictoGo 是一款面向英语学习者的听力与沉浸式学习 App。我们做它的起点很简单：英语能力不是靠技巧突然变好的，而是靠足够长时间地接触真实内容、理解完整句子、反复听读和持续输出。

一开始做 DictoGo，是受李笑来老师“1000 小时学习法”启发。语言学习没有捷径。真正有效的是高密度输入、有意识练习、句子级理解，以及足够多次的重复，让原本只是“看过”的表达慢慢变成自己能用的表达。

DictoGo 就是为这件事做的。

## 为什么做 DictoGo

很多人学英语，会经历一个很尴尬的阶段：单词认识不少，文章能看懂一点，视频也能听懂一些，但真正要开口或写出来时，还是卡住。

问题往往不是“词汇量太少”，而是缺少句子级练习：不知道一句话为什么这样组织，不知道表达如何在语境里使用，也没有把真正有用的词和句子反复听、反复练。

DictoGo 想帮你把这个闭环跑起来：从真实内容出发，逐句理解，反复听读，积累词汇和句子，再慢慢转成自己的输出能力。

## 从真实内容创建学习材料

你可以用多种方式创建学习文章：

- 粘贴英文文本或自己的学习材料
- 从母语文本出发，生成适合当前水平的英文内容
- 用 AI 创作定制主题文章
- 导入 PDF 文档
- 导入本地媒体或相册里的文件
- 从 Google Drive、阿里云盘、百度网盘等网盘导入
- 从 YouTube、哔哩哔哩等支持的视频链接导入

目标不是“多一个导入入口”，而是让你日常遇到的英语内容，都能变成可以逐句学习、反复听读和复盘的材料。

## 核心功能

### 句子级翻译

理解完整句子，而不是只查单个单词。适合那些“单词大概认识，但整句话还是不顺”的学习场景。

### Auto Echo 跟读

按句子重复播放和跟读，把听力输入变成口语练习。

### 生词本与背单词

从真实内容里收藏或导入词汇，整理到生词本，再通过单词训练复习。词汇不是孤立死背，而是和你读过、听过的内容绑定。

### 句库

把文章里的好句子加入句库，也可以手动添加长文本并自动拆句。句库适合反复听、跟读、复盘表达，把“见过的句子”变成“能用的表达”。

### AI 对话练习

通过 AI 对话，把理解过的表达真正用出来。

### 沉浸式听读

从更接近真实使用场景的内容中学习，而不是只靠割裂的教材例句。

## 适合谁

DictoGo 适合这些英语学习者：

- 能看懂一点、听懂一点，但开口容易卡住
- 想通过真实语境学习英语
- 需要整句解释，而不是只查词
- 想把词汇复习和真实内容绑定起来
- 想把好句子收集起来，反复听、跟读和输出
- 不想死背长词表，更希望稳定每天练
- 想找到从输入到输出的清晰路径
